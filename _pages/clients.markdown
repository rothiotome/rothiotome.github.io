I work with studios and indie developers as a technical consultant, helping navigate complex development challenges and build better games.

## Services

- **Technical Consultancy**: High-level guidance on game development strategy and implementation
- **Technical Interviews & Candidate Selection**: Conducting interviews and evaluating technical skills for game development teams
- **Twitch Integration Consultancy**: Designing and implementing Twitch features and live streaming integrations
- **Code Review & Architecture**: Feedback on code structure and development practices

I've worked with teams ranging from solo developers to established studios, adapting my approach to each project's specific needs.
